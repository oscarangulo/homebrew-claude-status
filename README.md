# homebrew-claude-status

Homebrew tap for [claude-status](https://github.com/oscarangulo/claude-status) — real-time token usage and cost monitoring for Claude Code.

## Install

```bash
brew tap oscarangulo/claude-status
brew install claude-status
```

## After install

```bash
claude-status install   # Configure Claude Code hooks
# Restart Claude Code
```

## Update

```bash
brew upgrade claude-status
claude-status update
```

## Uninstall

```bash
claude-status uninstall   # Remove hooks from Claude Code
brew uninstall claude-status
brew untap oscarangulo/claude-status
```
